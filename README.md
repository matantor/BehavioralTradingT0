# BehavioralTradingT0

## Working across computers

Minimal workflow:
- Clone the repo
- `npm install`
- `npm run dev`
- `npm run verify`
- Commit and push

Before committing:
- `git status`
- `git diff`
- `npm run verify`

Notes:
- Dependencies and build output are not committed (e.g., `node_modules/`, `dist/`).
